# Law 8: Graphics Must Contain Images

Every graphic (static AND animated) must contain at least one image as an element within the composition. The image is NOT the background — it is an element placed on top or beside text.

**Applies to:** Editor agent, Planner agent.
**Forbidden:** Blank background + text alone. Gradient/grid + text alone without an image.
**Required:** [designed background] + [image element(s)] + [text element(s)] + [shapes/decorative elements] = graphic.
