# Visual Types — Reference Video Analysis

There are **5 visual types** in the reference video. Every spoken line has one of these matching visuals.

## The 5 Types

### 1. Video Clip
Raw footage of matches, training, interviews, or any recorded moment.

### 2. Image
A still photo with video effects applied (zoom, pan, ken burns, filters, etc.). It has motion, but it is not a designed composition.

### 3. Static Graphic
A **designed composition** — images, icons, text, shapes, colors, all put together by a designer to communicate something visually. It is NOT "image with text." It is a designed piece. It is static (not animated).

### 4. Animation
An animated diagram that illustrates a concept that cannot be filmed — e.g. a formation board showing player movement, a tactical arrow, a data cascade. This is a visual explanation, not a graphic.

### 5. Animated Graphic
A graphic (same definition as Type 3 — a designed composition with images, icons, text, shapes) where **all internal elements are animated**. Logos fly in. Icons slide. Text types or fades. Shapes scale. Everything moves. Nothing appears statically. The whole composition comes to life through motion.

## Key Rules

1. **Every spoken sentence has a visual.** Nothing is said without something showing on screen.
2. **No blank background + text alone.** Every graphic is a designed composition, never a solid color or empty frame with text slapped on.
3. **Graphics can be static or animated.** Static graphic = designed but no motion. Animated graphic = designed and all elements move.
4. **Animations** explain concepts that footage cannot show (tactical shifts, cause-and-effect chains).
5. **Video clips** match exactly what the narrator is saying at that moment.
