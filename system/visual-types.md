# Visual Types — Reference Video Analysis

There are **5 visual types**. Every spoken line has one matching visual. Nothing is said without something showing on screen.

## The 5 Types

### 1. Video Clip
Raw footage of matches, training, interviews, or any recorded moment.

### 2. Image
A still photo with video effects applied (zoom, pan, ken burns, filters). It has motion, but it is not a designed composition.

### 3. Static Graphic
A **designed composition** — images, icons, text, shapes, colors, all put together by a designer to communicate something visually. It is NOT "image with text." It is a designed piece. It is static (not animated).

### 4. Animation
An animated diagram that illustrates a concept that cannot be filmed — e.g. a formation board showing player movement, a tactical arrow, a data cascade. This is a visual explanation, not a graphic.

### 5. Animated Graphic
A graphic (same definition as Type 3 — a designed composition with images, icons, text, shapes) where **all internal elements are animated**. Logos fly in. Icons slide. Text types or fades. Shapes scale. Everything moves. Nothing appears statically.

## Key Rules

1. **Every spoken sentence has a visual.** Nothing is said without something showing on screen.
2. **No blank background + text alone.** Every graphic is a designed composition, never a solid color or empty frame with text.
3. **Every graphic (static AND animated) MUST include at least one image as an element.** The image is NOT the background — it is an element within the composition. Multiple images can be used. A graphic without an image is WRONG.
4. **No reusing images across graphics.** Each graphic uses a DIFFERENT image.
5. **No watermarked images.** After generating any graphic, VLM-check for watermarks before committing.
6. **Graphics can be static or animated.** Static = designed but no motion. Animated = designed and all elements move.
7. **Animations** explain concepts that footage cannot show (tactical shifts, cause-and-effect chains).
8. **Video clips** match exactly what the narrator is saying at that moment.

## Transitions

- Transitions are **narrative-based**, not visual-based.
- A transition happens when the STORY moves to a new point.
- NOT every clip change is a transition. Clip changes within the same narrative point are just cuts.
- Every transition gets a transition sound effect + visual transition effect (crossfade, dissolve, etc.)

## Authority Clips

Pattern: ~30-40 seconds of narration → 10-15 second authority clip (pundit/coach speaking).

- The authority clip REINFORCES what the narrator just said.
- Before the authority clip, there is a TRANSITION + SFX.
- During the authority clip: narration audio is MUTED. Authority clip's own audio plays at full volume.
- Background music continues underneath (ducked).
- If no authority clip is available, flag for the user — do NOT substitute with regular B-roll.
